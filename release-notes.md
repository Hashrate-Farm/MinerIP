## MinerIP v1.0.0 — ASIC Miner Network Scanner

Lightweight Windows tool that scans your local network and finds all cryptocurrency miners.

### Features
- One-click subnet scan — miners appear in real time
- Identifies brand & model (Antminer, Whatsminer, Avalon, Bitaxe, IceRiver, Goldshell, and 20+ more)
- Shows live hashrate, temperature, fan speed, and power consumption
- Clean web UI opens automatically in your browser
- Single portable `.exe` — no installation required

### Download
1. Download **MinerIP.zip** below
2. Extract and run **MinerIP.exe**
3. A browser tab opens with the scanner UI
4. Click **Start Scan**

### Windows SmartScreen
Because this is a new unsigned executable, Windows SmartScreen may show a warning. Click **"More info"** → **"Run anyway"**. The full source code is available for review in this repository.
